## Description
This assignment implements Dijkstra's Algorithm in Java to find the shortest path from a starting node to all other nodes in a weighted graph. The algorithm selects the node with the smallest distance, updates its neighbors, and repeats until all nodes are processed. It uses arrays and priority queues for efficiency.

## How to Run
1. Open the project in a Java IDE (Eclipse, IntelliJ).  
2. Compile all Java files.  
3. Run `Main.java` to see the shortest path results.

## Discussion
The program correctly calculates shortest distances for the sample graphs.  
- Works well for graphs with non-negative weights.  
- Priority queue improves performance by selecting the smallest-distance node efficiently.  
- Shows the importance of arrays and queues in graph algorithms.  

This assignment helped understand graph traversal, shortest path computation, and practical implementation in Java.
